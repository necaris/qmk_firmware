# Introduction

Simple tweaks to the default ErgoDox EZ layout to make things slightly better
for my personal use (as an Emacs user but otherwise pretty vanilla QWERTY
typist).

## Keymap 0: Basic layer
```

 |--------+------+------+------+------+-------------|           |------+------+------+------+------+------+--------|
 | Tab    |   Q  |   W  |   E  |   R  |   T  |LShift|           |RShift|   Y  |   U  |   I  |   O  |   P  |   \    |
 |--------+------+------+------+------+------|      |           |      |------+------+------+------+------+--------|
 | LCtrl  |   A  |   S  |   D  |   F  |   G  |------|           |------|   H  |   J  |   K  |   L  |; / L2|   '    |
```
